# Life Partner Proposal Interactive Page 💍

A fun, heartfelt, and interactive web page to propose to your life partner! In this cute proposal to "Fizu", the "Yes" button grows bigger every time the "No" button is clicked — complete with falling hearts, cute GIFs, background music, and playful toast messages.

Built with pure HTML, CSS, and JavaScript.

**Live demo:** [Fardeen2812.github.io/Purposal](https://Fardeen2812.github.io/Purposal) *(Update this link if your repository name changes)*

---

## Want to Use This to Impress Your Partner?

You can easily adapt this template for your own proposal. 

### Option 1: Fork It and Make It Yours

Want your own version you can customize with your partner's name and your own pictures? Follow these steps:

#### 1. Fork the Repo
- Go to [github.com/Fardeen2812/Purposal](https://github.com/Fardeen2812/Purposal)
- Click the **Fork** button in the top-right corner
- This creates a copy of the repo under your GitHub account

#### 2. Enable GitHub Pages
- In your forked repo, go to **Settings** (the gear icon tab)
- In the left sidebar, click **Pages**
- Under **Source**, select **"Deploy from a branch"**
- Under **Branch**, select **`main`** and **`/ (root)`**
- Click **Save**

#### 3. Wait ~2 Minutes
- GitHub will build and deploy your site
- Your site will be live at:
  ```
  https://yourusername.github.io/Purposal
  ```
  *(Replace `yourusername` with your actual GitHub username)*

#### 4. Personalize It
- Edit `index.html` to change the proposal question, name, or GIFs
- Edit `yes.html` to change the celebration message and response
- Edit `script.js` to tweak the guilt-trip messages, tease toasts, button behavior, or GIFs
- Edit `style.css` to change colors, fonts, or animations
- Swap out the music file in the `music/` folder with your own special song

> **Tip:** You can edit files directly on GitHub by clicking on a file and hitting the pencil (edit) icon. No need to clone anything locally if you just want quick text changes.
